## KDDI

### セキュリティ・利用ポリシー

| 項目 | 内容 |
|------|------|
| **外部AI直接利用** | ChatGPT等の直接業務利用は**非推奨**。セキュリティリスク（情報流出・学習利用）を明確に認識 |
| **代替手段** | Azure OpenAI Service を採用し、**自社専用環境「KDDI AI-Chat」**を構築。入力データがAIの学習に使われない仕組み |

| **社内展開規模** | 2023年5月〜、社員約**1万人**が「KDDI AI-Chat」を業務利用開始 |
| **部門横断体制** | 2023年5月8日〜、全社横断の生成AI利活用推進体制を構築 |

**ポイント**: NECの「NGS経由のみ」と同様、KDDIも**外部AIの直接利用ではなく、自社閉域環境経由**のアプローチを採用。Azure OpenAI Serviceベースで、セキュリティと利便性を両立している。

### 活用事例・成果

#### 営業準備時間の74%削減（ミドル業務革新）
- Azure OpenAI Serviceを活用し、法人営業の情報収集・資料作成を効率化
- 1ユーザーあたりの資料作成における**情報収集時間を約74%削減**
- 80%のユーザーが情報収集時間の削減を実感
- 背景：法人向け営業の顧客接触時間は業務全体の約23%に留まっていた課題を解決

#### 法人営業部門全体の成果
- 生成AIツール実装から1年余りで、**年間46,939時間**削減
- 営業1人あたり**月10時間**の削減効果
- 「AIを使う新人ほど早く結果を出す」傾向を確認
- ハイパフォーマー営業の会話をAIで分析し、ベストプラクティスを展開

#### KDDI Conata Data Agent（2025年9月〜）
- フライウィールの情報検索技術＋KDDIの閉域網を組み合わせた社内文書検索AI
- 週1人あたり8時間の社内資料検索・分析・資料作成の作業負荷を**約3割削減**
- 2025年7月18日〜法人営業部門に先行導入。導入ユーザーの**8割以上**が調査時間短縮を実感

#### 広告クリエイティブ生成AI
- 広告クリエイティブ生成AIシステムをゼロから構築
- **工数50%削減**を達成

#### ELYZA Works with KDDI（2025年9月〜）
- 専門知識不要でAIアプリ開発が可能なプラットフォーム
- KDDI社内で約**100種類のAIアプリ**を先行開発し、好事例をセット提供

## ソフトバンク

### セキュリティ・利用ポリシー

| 項目 | 内容 |
|------|------|
| **初期対応（2023年3月）** | ChatGPT等の外部AI利用を**制限**。機密情報の入力を禁止する通知を全社員に発出 |
| **制限理由** | OpenAIの無償版は入力データをAI学習に利用する可能性 → 開発中の製品情報等が流出するリスク |
| **現在の方針** | 自社管理下のセキュアな環境で生成AIを提供。**Azure OpenAI Service**ベースの「dailyAI」等を活用 |
| **セキュリティ機能** | ユーザー認証（Microsoft Entra ID）、IP制限、個別環境提供、アップロードファイルの自動消去、生成AIの学習からの除外 |
| **閉域接続** | 生成AIパッケージ（MSPサービス）にて閉域接続に対応。企業独自のセキュリティ基準を満たす環境構築が可能 |

**ポイント**: ソフトバンクもNEC・KDDIと同様、**外部AIの直接利用は制限し、自社管理環境経由**で提供するアプローチ。ただし社内での利活用推進は他社以上に積極的。

### 社内普及・組織

| 項目 | 内容 |
|------|------|
| **トップコミットメント** | 宮川潤一社長が直接旗振り。「AIへの静観は退化を意味する」と全社に号令 |
| **AX（AI Transformation）** | 全社戦略として「AX」を掲げ、通信事業→次世代社会インフラ企業への転換を推進 |
| **AI推進子会社** | **Gen-AX株式会社**（企業向けAX支援SaaS）、**SB Intuitions株式会社**（国産LLM開発） |
| **3本柱の推進体制** | ①AI利活用基盤の整備 ②生成AIの学習プログラム提供 ③活用促進施策 |
| **AI資格保有率** | AI関連資格保有者は全社員の約**13%** |
| **受賞歴** | 「**GenAI HR Awards 2025**」企業セクター・大手企業の部で**グランプリ受賞** |

#### AIエージェント祭（2025年夏）
- **ミッション**: 社員1人あたり100個のAIエージェントを作成
- **期間**: 約2カ月半（10週間）
- **成果**: 全社で**250万超のAIエージェント**を作成（約2万人の社員が参加）
- **狙い**: 質よりまず「量」による**AI文化醸成**
- **効果**: 参加社員の**90%**が取り組みを肯定的に評価、**80%**が「今後のAI活用を具体的にイメージできるようになった」
- 宮川社長が2025年6月の朝礼で直接号令 → 6週間で90万本 → 最終的に250万超

### NECへの示唆

1. **セキュリティアプローチは業界共通**: 3社とも「外部AI直接利用は制限し、自社管理環境経由」で一致。NECのNGS経由アプローチは業界標準に沿っている
2. **差がつくのは「普及の速度と深度」**: ソフトバンクの「AIエージェント祭」のような全社巻き込み施策、KDDIの「7割以上が業務活用」という浸透率が競争力の源泉
3. **トップコミットメントの重要性**: ソフトバンク宮川社長の直接関与は、推進速度に直結している
4. **独自LLM vs 外部LLM活用**: KDDIはELYZA（オープンモデル活用）、ソフトバンクはフルスクラッチ開発と、戦略は二極化。NECはどちらの路線を取るかが戦略的判断ポイント
5. **数値で語れる成果の重要性**: 両社とも「74%削減」「250万エージェント」等の具体的数値を対外的に発信し、ブランディングに活用
