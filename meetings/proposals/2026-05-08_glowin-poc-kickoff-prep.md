## 想定される質問と回答方針

| 想定Q | 回答方針 |
|-------|----------|
| 投稿開始はいつから？ | アカウント設計書合意後（5月下旬目処）に試運用開始、6月から本格運用 |
| 既存のCFO EDGEとの役割分担は？ | Xで反応が出たテーマ→CFO EDGE記事化／CFO EDGE記事の結論パート→X単発化、の双方向ループ |
| 採用アカウントは？ | 本案件のスコープ外（ブランド認知優先）。次フェーズで分離検討 |
| 投稿の最終承認権は誰？ | 本田さんが Slack で最終承認。例外時は佐野さんに即時エスカレーション |
| 公認会計士として発信する範囲は？ | 守秘義務に抵触しない範囲。グローウィンの直接案件の固有名詞は出さない／一般化した実装ログ／海外事例のキュレーション中心 |
| KPIは？ | 平均視聴時間・保存数・プロフィールクリック・指名検索数。**フォロワー数とイイネは見ない** |
