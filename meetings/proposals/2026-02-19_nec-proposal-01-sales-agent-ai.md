# NEC法人営業部門向け セールスエージェントAI 提案書

**提案日**: 2026年2月19日
**提案者**: 一般社団法人ビジネスAI推進機構（BAAO） / 合同会社Unson（雲孫）
**宛先**: 日本電気株式会社 法人営業部門 田岡様

## エグゼクティブサマリー

NECの法人営業チームが抱える**情報収集・提案書作成・CRM入力**等の非効率を、既存のMicrosoft 365 Copilot基盤を最大活用した「セールスエージェントAI」で解消します。

競合他社ではKDDIが営業準備時間74%削減（年間4.7万時間）、NTTデータがRFP対応6割短縮を実現しています。NECが同等の施策を先行実装することで、**営業生産性の飛躍的向上**と**CIOへのプレゼンス強化**を同時に達成できます。

BAAAOは経産省AIガバナンスガイドライン執筆協力の知見を持ち、Unsonが実装を担当。**研修→コンサル→実装の一気通貫体制**で、大手コンサルの1/10の費用感で確実に成果を出します。

## 1. 課題定義：NECの営業現場が直面する非効率

### 1.1 営業担当者の時間配分の現実

一般的に法人営業担当者の業務時間のうち、**顧客との対面・商談に費やせる時間は全体の約30%**に過ぎません。残り70%は以下のような間接業務に消費されています。

| 間接業務 | 推定所要時間/件 | 課題 |
|---------|---------------|------|
| **顧客・業界情報の収集** | 2〜4時間 | 社内外のソースが分散、属人的な情報網に依存 |
| **提案書・見積書の作成** | 4〜8時間 | テンプレートはあるが顧客別カスタマイズが手動 |
| **CRM/SFA入力** | 30分〜1時間/日 | 商談後の記録が後回しになり、データ品質が低下 |
| **社内調整・承認プロセス** | 1〜3時間 | メール・Teams検索で過去事例を探す非効率 |
| **会議準備・議事録作成** | 1〜2時間 | 手動メモ→整形→共有の繰り返し |

### 1.2 NEC固有の環境制約

- **Copilot（M365）は利用可能** → 既存投資を活かす土台がある
- **外部AI直接利用は禁止（NGS経由のみ）** → セキュリティポリシー準拠が必須
- **Kotomi（NEC社内AI）の営業活用が進んでいない** → 現場に刺さるユースケース設計が鍵
- **CAIO的役職・横串AI組織を2025年末に新設** → AI推進の社内モメンタムは存在する

### 1.3 放置した場合のリスク

- 競合（KDDI・富士通・NTTデータ）がAI営業で先行し、提案スピードで後れを取る
- CAIO組織の設立効果が営業現場まで届かず、「AI推進しているのに成果が見えない」状態に
- 優秀な営業人材が非効率な業務に疲弊し、離職リスクが上昇

## 2. ソリューション概要：Copilot活用セールスエージェントAI

### 2.1 アーキテクチャ設計思想

NEC既存環境（M365 + Copilot + NGS）を最大限活用し、**新たなSaaSを導入せずに**営業プロセスを自動化します。

│  │ Outlook  │   │  Teams   │   │  SharePoint   │  │
│  │(メール)   │   │(会議/Chat)│   │(文書・ナレッジ) │  │
│  └────┬─────┘   └────┬─────┘   └──────┬────────┘  │

### 2.2 セールスエージェントAI：5つのコア機能

#### 機能① 顧客インテリジェンス自動収集

**Before**: 営業担当が手動で社内文書・IR情報・ニュースを検索（2〜4時間）
**After**: 商談前に自動で顧客ブリーフィングシートを生成（5分）

- SharePoint内の過去提案書・契約情報をCopilotが横断検索
- 顧客の最新IR・ニュースリリースを要約（NGS経由で安全に取得）
- 過去の商談履歴・キーパーソン情報をCRMから自動抽出
- **出力**: 1ページの「商談準備シート」（Teams/Outlook経由で自動配信）

#### 機能② 提案書ドラフト自動生成

**Before**: テンプレートから手動カスタマイズ（4〜8時間）
**After**: 顧客情報+製品情報から初稿を自動生成（30分で完成度70%）

- Copilot Studio上にカスタムエージェントを構築
- 顧客の業界・課題・予算規模に応じたテンプレート自動選択
- SharePoint内の成功提案書をRAG（検索拡張生成）で参照
- 営業担当は生成された初稿をレビュー・微調整するだけ

#### 機能③ 商談議事録・ネクストアクション自動化

**Before**: 手動メモ → 議事録作成 → CRM入力 → フォローアップメール（1〜2時間）
**After**: Teams会議から自動生成（即時）

- Teams Copilotの議事録機能を営業向けにカスタマイズ
- 議事録から「決定事項」「課題」「ネクストアクション」を自動抽出
- CRM/SFAへ商談記録を自動連携
- フォローアップメールのドラフトを自動生成（Outlook Copilot）

#### 機能④ CRM/SFAデータ自動入力・品質向上

**Before**: 商談後に手動入力、入力漏れ多発（30分〜1時間/日）
**After**: メール・Teams・カレンダーから自動補完（ほぼゼロ）

- Outlook/Teamsでの顧客コミュニケーションからCRMフィールドを自動推定
- 商談ステージ・確度・次回アクション日を自動更新
- 入力漏れアラート（週次レポート自動生成）

#### 機能⑤ 営業ナレッジアシスタント（FAQ/ベストプラクティス）

**Before**: 先輩社員に口頭で確認、属人的ノウハウ
**After**: Copilot上でチャット形式で即座に回答

- 社内の成功事例・失敗事例をSharePointから学習
- 「この業界のお客様にはどのソリューションが刺さるか？」に即答
- 新人営業の立ち上がり期間を大幅短縮

## 3. 期待効果：数値で見るインパクト

### 3.1 競合他社の実績ベンチマーク

| 企業 | 施策 | 効果 |
|------|------|------|
| **KDDI** | Copilot活用（営業含む全社） | 営業準備時間 **74%削減**、年間 **4.7万時間** 削減 |
| **富士通** | GitHub Copilot導入（開発） | 作業時間 **20%以上** 短縮 |
| **NTTデータ** | AIエージェント（提案対応） | RFP対応 **6割短縮** |

### 3.2 NEC法人営業への適用試算

営業担当者100名規模の部門で試算：

| 指標 | 現状 | 導入後（推定） | 削減率 |
|------|------|-------------|--------|
| 商談準備時間 | 3時間/件 | 45分/件 | **75%削減** |
| 提案書作成時間 | 6時間/件 | 2時間/件 | **67%削減** |
| CRM入力時間 | 45分/日 | 5分/日 | **89%削減** |
| 議事録作成 | 1時間/件 | 5分/件（自動生成+確認） | **92%削減** |

**年間効果（100名規模試算）**:

| 項目 | 試算値 |
|------|--------|
| 創出時間 | **年間約3〜4万時間**（1人あたり年間300〜400時間） |
| 顧客対面時間増加 | 現状比 **+40〜50%** |
| 新規商談機会 | **+20〜30%**（創出時間を新規開拓に転換） |
| 受注率改善 | **+10〜15%**（提案品質向上・レスポンス速度向上） |

### 3.3 定性効果

- **CIOへのプレゼンス向上**: 「NEC自身がAI営業を実践している」という説得力
- **横串AI組織の成果可視化**: CAIO配下の具体的成功事例として社内発信可能
- **採用競争力**: 先進的な営業DX環境を訴求
- **データドリブン経営**: CRMデータ品質向上→正確なフォーキャスト
