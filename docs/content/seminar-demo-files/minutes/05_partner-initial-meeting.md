# 顧客面談議事録 — 株式会社サンクスフィールド（パートナー候補との初回）

**日時**: 2026/05/11（月）13:00-14:30
**面談者**: 営業課 小林（弊社） / 事業開発部 西村部長（サンクスフィールド様）
**形式**: 弊社オフィスにて対面

## 議事内容

### 1. サンクスフィールド様の事業概要
- 設立 12 年、社員 95 名、年商 ¥18 億
- 主力事業: 中小企業向け業務システムのスクラッチ開発
- 顧客基盤: 製造業・物流業の中小企業 200 社超
- 強み: 業界特化の業務知識、長期保守契約による安定収益

### 2. 連携の打診内容
- サンクスフィールド様の顧客基盤に、弊社プロダクトを販売したい
- 形式: パートナー契約（販売代理店 / レベニューシェア型）
- レベニューシェア比率: 業界相場の 20-30% を想定

### 3. 西村部長の関心
- 「自社で AIプロダクトを作る余裕はないが、顧客から要望は来ている」
- 「弊社のような既存システムベンダーが AI を扱えるようになる手段が欲しい」
- 「単なる販売代理よりも、自社ブランドで売れる OEM 的な形に興味」

### 4. 弊社からの提案
- まずは 3 ヶ月のパイロット期間を提案
- パイロット中: サンクスフィールド様の顧客 5 社にトライアル提供
- パイロット後の本契約条件は実績見て決定
- OEM については、6 ヶ月以上の関係構築後に検討

### 5. 西村部長の懸念
- 「弊社の顧客は IT リテラシーが低い。サポート体制が回るか」
- 「価格設定が弊社の顧客層に合うか」（弊社の月額 ¥30万〜 は中小企業には高い）
- 「弊社エンジニアが理解できる範囲のプロダクトか」（技術ドキュメントの質）

### 6. ブロッカー / 上司判断必要事項
- パートナー契約の枠組み: 役員レベルの判断が必要
- 中小企業向け価格プラン（月額 ¥10万〜 程度）の新設可否 → 経営判断
- パイロット期間中のサポート体制（誰がコストを持つか） → 上司判断

### 7. 次回の動き
- 5/20 までに弊社内で「パイロット提案書」をまとめて提出
- 5/27 に西村部長 + サンクスフィールド経営陣 同席の2回目ミーティング
- 役員レベルの面談を 6 月上旬に設定する方向

## 小林所感

サンクスフィールド様の顧客基盤 200 社は魅力。一方、価格と技術ドキュメントの壁が高い。
OEM/パートナー戦略は弊社にとって新領域なので、社内整理が必要。
このタイミングで「中小企業向け価格プラン」を立てる選択肢はあるか、経営判断を仰ぎたい。
