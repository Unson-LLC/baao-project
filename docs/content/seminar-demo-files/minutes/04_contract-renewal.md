**形式**: みらい銀行様 本店 ミーティングルーム

## 議事内容

### 1. 現契約の状況
- 契約: 2024年6月開始、初年度 → 2年目に入る更新タイミング
- 年額: ¥3,200,000（月額換算 ¥266,000）
- 利用範囲: 行員 220 名向けの社内文書検索AI

### 2. 更新条件の論点
- 弊社提案: 同条件で更新（年額 ¥3,200,000）
- みらい銀行様要望: **20% 値下げを希望**（年額 ¥2,560,000）
- 木村次長の根拠: 「予算圧縮の指示が出ている」「競合B社から30%安い提案を受けている」

### 3. 利用状況の事実
- 月間アクティブユーザー率: 78%（業界平均 60% を超える健全な数字）
- 行員アンケートでの満足度: 4.2/5.0
- 削減効果: 1人あたり月 4.5 時間（試算で年間 ¥18,000,000 相当の人件費削減）
- ROI は十分プラス、これは木村次長も認識

### 4. 弊社からの再提案
- 価格そのまま、新機能（音声入力対応）を無償で追加する案を提示
- もしくは、利用ユーザー数を 220 → 280 に増やして単価を下げる案
- 「20% ストレート値下げ」は弊社的にも厳しいことを伝達

### 5. 松本さん（経営企画）の発言
- 「ROI は理解しているが、経営から数字としての圧縮指示が出ている」
- 「ストレート値下げか、付加価値追加かのどちらかが欲しい」
- 「機能追加なら、議事録AIも追加できないか」（新規要望）

### 6. ブロッカー / 上司判断必要事項
- 議事録AI 追加提案の可否 → 上司・プロダクト判断
- 20% 値下げを受け入れるか、付加価値で勝負するか → 営業マネジメント判断
- 競合B社の提案内容を確認したい → 競合インテリジェンス

### 7. 次回の動き
- 5/16 までに「価格据置 + 議事録AI追加」の見積を提出
- 5/23 に木村次長 + 松本さん + 弊社管理職 同席のオンライン提案
