# 顧客面談議事録 — 株式会社ブルースカイテック（新規提案）

**日時**: 2026/05/09（土）10:30-12:00
**面談者**: 営業課 佐々木（弊社） / 開発本部 加藤本部長 + 情報システム室 山本さん（ブルースカイテック様）
**形式**: ブルースカイテック様 本社にて対面

## 議事内容

### 1. 先方の現状課題
- 社員数 380 名、ITサポート専任は 3 名
- 社内ヘルプデスクへの問い合わせが月 800 件超、ほぼ手作業対応
- パスワードリセット・VPN接続不具合 などの定型問い合わせが全体の 60%
- 「定型対応を自動化したい」が加藤本部長の最大の関心

### 2. 弊社からの提案
- AIチャットボット + ナレッジ自動更新システム
- 想定削減効果: 月 480 件の問い合わせを自動化（60%削減目標）
- 初期費用 ¥800,000、月額 ¥120,000

### 3. 加藤本部長の反応
- 「数字感はちょうどいい。問題は導入スピード」
- 6月中の稼働を希望（理由: 7月の組織改編前に成果を出したい）
- 弊社の標準納期は 8 週間と説明したが、「6 週間で何とかならないか」と要望

### 4. 山本さん（情シス）の懸念
- 既存ナレッジが Confluence と Notion に分散している
- データクレンジングに想像以上の工数がかかりそう
- 弊社側でクレンジングを巻き取れるか確認したいとのこと

### 5. ブロッカー / 上司判断必要事項
- 6 週間納期は標準外。社内リソース確保が必要 → 上司判断
- データクレンジングの巻き取り範囲・追加費用 → 上司判断
- 加藤本部長が「相見積もりを取る」と明言。競合候補は A社 と B社

### 6. 次回の動き
- 5/15 までに正式見積（6 週間納期パターン含む）を提出
- 5/20 に加藤本部長 + CTO 同席のオンライン提案会

## 佐々木所感

受注確度は 60-70%。価格より納期がボトルネック。
データクレンジングを巻き取れれば、競合に対して明確な優位性が出る。
ただし、社内リソース的に Q2 後半は別案件と並走になるので、上司の判断が要る。
